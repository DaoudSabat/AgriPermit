# AgriPermit - No Docker Setup

## Start Services
```
start-local.bat
```

This will start:
- API: http://localhost:8000
- Web: http://localhost:5173

## Mobile App
```
cd AgriPermitMobile
npm install
npx expo start
```

Scan QR code with Expo Go app on your phone

## Stop Services
```
stop-local.bat
```
